# vfox-sentry-cli

[![CI](https://github.com/jbox-web/vfox-sentry-cli/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-sentry-cli/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [sentry-cli](https://github.com/getsentry/sentry-cli)

## Install

```
mise plugin install sentry-cli https://github.com/jbox-web/vfox-sentry-cli.git
```
